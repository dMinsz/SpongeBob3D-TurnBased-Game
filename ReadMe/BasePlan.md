# 기본 기획

기간 : 2일 
장르 : 3d 턴제 / JRPG Style

**특징:**
- 심볼릭 인카운터 게임

- 속성은 없음

- 최대 플레이가능 캐릭터 3명

- 한번에 나오는 적 4기 , Wave 개념 으로 더추가 가능

## 플레이 방식

정해진 맵에서 돌아다니는 몬스터 개체를

기습 하거나 피습 당해서 배틀 을 시작하여

배틀을 승리, 패배 하면서 

게임을 진행


## 기본 배틀 방식

특정 맵에 몬스터가 돌아다니고

그 몬스터를 기습 하면 (공격을 먼저하면)

플레이어 턴 으로 배틀 시작, 아니면 적의 턴으로 배틀 시작


적, 플레이어 기본 행동

공격, 스킬 사용, 도망가기(적은 못함, 캐릭터는 확률)


## 기본 시나리오


스폰지밥이 비키니시티가 기계들에 습격을 받아서

기계들을 잡고다니고 이후 플랑크톤 기계를 파괴하면 끝

## 최소 구현

스폰지밥 배틀 

몬스터 2-3기 정도 배틀

보스 몹인 로보 플랑크톤 배틀 후 종료
